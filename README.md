# Sistema de Gestión - Clínica Quiropráctica 🏥

Este proyecto es un sistema integral para la gestión de pacientes, citas y tratamientos en una clínica quiropráctica, desarrollado como parte del curso de **Análisis y Diseño de Sistemas (ADS)**.

## 🚀 Características
- **Gestión de Pacientes:** Registro, edición y consulta de historias clínicas.
- **Control de Citas:** Programación y seguimiento de sesiones.
- **Módulo de Tratamientos:** Registro detallado de procedimientos aplicados.
- **Interfaz Intuitiva:** Desarrollado con lógica orientada a facilitar el flujo de trabajo clínico.

## 🛠️ Tecnologías utilizadas
- **Lenguaje:** Java ☕
- **Lógica:** Programación Orientada a Objetos (POO).
- **Entorno: Eclipse y SpringTools** .

## 📋 Metodología (ADS)
Para el desarrollo de este sistema se aplicaron conceptos de:
- Diagramas de Casos de Uso.
- Arquitectura de software.
- Diseño de interfaces centradas en el usuario.
